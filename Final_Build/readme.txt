=========================================
PROGRAM DESCRIPTION
=========================================
MultiDraw is an extended version of ObjectDraw that permits multiple users on different computers to

collaborate across a network on a shared drawing. MultiDraw permits remote users to view a session in

real time and allows the owner of a drawing to transfer control to another user, thereby allowing the

remote user to manipulate and modify the drawing. Other connected users can continue to view these

changes in real time. Drawing canvas sessions can be saved and reopened for display and modification.
MultiDraw allows the user to add and share drawing tools without recompiling the application.
 

=========================================
